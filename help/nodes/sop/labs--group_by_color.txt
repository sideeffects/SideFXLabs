""" Creates a group based on Point Colors """

This node groups points based on Point Colors, with an interface similar to a colorkey, where you select your main color and a fuzziness threshold. 

@parameters
    Group Type:
        Where is the Cd Attribute?
    Group Name:
        What the output group should be called 
    Color:
        Key Color to select
    Threshold:
        Fuzziness factor around the color
